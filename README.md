# Login Token Skeleton

| :----------: | :-----------------------------: |
|    Spring    |    Spring Boot 2.3.4.RELEASE    |
|      OS      |            Mac OS X             |
|   개발 툴    | Intellij IDEA Ultimate 2020. 01 |
|     JDK      |              JDK 8(>=8)              |
| 데이터베이스 |               H2, MySQL                |
|   빌드 툴    |          gradle-6.6.1           |
